# SISTEMA CRM - GRUPO MADOO

O sistema de CRM tem como principal objetivo otimizar e agilizar a criação de orçamentos, contribuindo diretamente para o aumento do volume de vendas do Grupo Madoo. Pensado para atender o dia a dia da célula comercial, ele oferece aos vendedores uma plataforma intuitiva e centralizada que facilita o registro de clientes, produtos, interações e negociações.

Além disso, o sistema também incorpora a atuação dos analistas, que desempenham um papel estratégico no processo comercial. Eles podem acompanhar o desempenho das equipes, monitorar o andamento dos orçamentos e realizar a validação ou reprovação das propostas, garantindo que cada negociação siga os padrões e diretrizes internas.

Dessa forma, o CRM não apenas acelera o trabalho dos vendedores, mas também assegura maior controle, organização e confiabilidade no ciclo de vendas como um todo — desde o primeiro contato até o fechamento do orçamento.
